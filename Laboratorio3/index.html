Proporcionan comunicacion entre las acciones solicitadas u operaciones entre cliente y servidor

Se utiliza el metodo GET el cual solicita los datos de la pagina web para poder desplegarlos.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

El error 404 se debe a que no se ha encontrado la pagina web por algunas razones por lo que el desarrollador debe investigar si se trata del lado del desarrollo o por parte de usuario como error al escribir la URL de la pagina para poder corregirlo. En la mayoria de los casos el desarrollador no deberia resolverlo ya que lo mas probable sea algun error no relacionado al desarrollo del sitio.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Este error proviene del srevidor por lo que en este caso el desarrollador si deberia intervenir para corregir el error.

HTML 5 presenta mayor compatibilidad con browsers modernos mientras que en html 4 puede no funcionar debidamente. HTML 5 soporta elementos audiovisuales HTML 5 es mas rapido en responder a las peticiones del usuario (tambien en disp moviles) HTML 5 presenta mejoras al implementar formularios en sitios web antes solo permitia user y password ahora tampoco depende de una tecnologia externa para su funcionamiento
