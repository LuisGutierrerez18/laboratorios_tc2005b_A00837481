Preguntas

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguje orientado a objetos es decir que se basa en crear clases con atributos a un objeto (puede ser fisico). Tambien es muy utlizado para crear apps y videojuegos. Java Script es un lenguaje de scripting lo que significa que interpreta el código y lo ejecuta en el momento. JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

getDate(), setDate(), getDay(), getYear(), getHour()

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

pop(), push(), arrayLength(), arrayConcat(), arrayAt()

¿Cómo se declara una variable con alcance local dentro de una función?

Se declara con la palabra "let" or "var"

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Que pueden utilizar en multiples metodos ya que no esta limitada a solo ser utilizada en una sola funcion. Nos ayuda a optimizar el uso de variables.
